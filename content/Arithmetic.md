## $\mathbb N$ Natural Number
This is the number you use to count whole objects. 1,2,3,4,5,... The ... means to continue this pattern forever. Some people include 0, however many doesn't.
We can write $\mathbb N_0$ to represent the natural number including 0

## Addition
Addition is to process of continued counting. Start counting from the first number by the second number. Consider these two questions questions for the next two examples.

1. Does both expression result in the same number?
2. What 's the logical difference between both expression?

- 2 + 3 and 3 + 2
> [!info] We are counting the same total. Counting 2 things first then count 3 more things is the same as counting 3 things first then count 2 more things. So they are the same.
- (2 + 3) + 5 and 2 + (3 + 5) 
> [!info] We are counting the same total. So they are the same.

5 + 0
> [!info] Adding zero means to count nothing more

0 + 5
> [!info] This means to start counting from nothing
